歡迎來到資安威脅與攻擊篇。資安不僅是專家的事，更是開發者的日常責任。這集我們將認識各種常見的威脅與攻擊手法，知己知彼，才能百戰百勝。

首先，我們從系統中最基本的弱點概念開始，了解駭客如何尋找機會。

弱點
Vulnerability
We found a critical vulnerability in the authentication code.
我們在驗證程式碼中發現了一個嚴重弱點。

漏洞利用
Exploit
Hackers used an exploit to gain unauthorized access to the system.
駭客使用漏洞利用手段來獲取系統的未授權存取。

零時差攻擊
Zero-day
A zero-day vulnerability is a serious threat with no known fix yet.
零時差弱點是一個尚未有修復方案的嚴重威脅。

資料外洩
Data Breach
The company suffered a massive data breach involving user emails.
該公司遭受了涉及使用者信箱的大規模資料外洩。

惡意軟體與欺騙手段是攻擊者常用的武器，這些名詞你一定聽過。

惡意軟體
Malware
Malware 是 Malicious Software 的縮寫
Scan the system regularly to detect any malware.
定期掃描系統以偵測任何惡意軟體。

勒索軟體
Ransomware
Ransomware encrypted all our files and demanded payment.
勒索軟體加密了我們所有的檔案並要求贖金。

後門
Backdoor
The trojan installed a backdoor for remote access.
特洛伊木馬安裝了一個後門以進行遠端存取。

釣魚攻擊
Phishing
Be careful of phishing emails asking for your password.
小心要求你提供密碼的釣魚郵件。

社交工程
Social Engineering
Social engineering manipulates people into revealing secrets via phone or email.
社交工程透過電話或郵件操弄人們洩露機密。

針對應用程式邏輯的攻擊，往往最難以防範，開發者必須特別留意。

注入
Injection
Prevent SQL injection by using parameterized queries.
使用參數化查詢來防止 SQL 注入。

跨站腳本
XSS
XSS 是 Cross-Site Scripting 的縮寫
Escape output content to prevent XSS attacks.
轉義輸出內容以防止 XSS 攻擊。

跨站請求偽造
CSRF
CSRF 是 Cross-Site Request Forgery 的縮寫
Use anti-CSRF tokens to verify the request origin.
使用抗 CSRF Token 來驗證請求來源。

暴力破解
Brute Force
The attacker tried a brute force attack to guess the password.
攻擊者嘗試對密碼進行暴力破解猜測。

提升
Escalation
Privilege escalation allows a user to gain higher access rights.
特權提升允許使用者獲取更高的存取權限。

在網路層級，攻擊者試圖癱瘓服務或竊聽通訊，常見手法如下。

阻斷服務攻擊
DDoS
DDoS 是 Distributed Denial of Service 的縮寫
The server is down due to a massive DDoS attack.
伺服器因為大規模的 DDoS 攻擊而當機。

中間人攻擊
Man-in-the-Middle
Public Wi-Fi is vulnerable to Man-in-the-Middle attacks.
公共 Wi-Fi 容易遭受中間人攻擊。

欺騙
Spoofing
IP spoofing hides the attacker's true identity.
IP 欺騙隱藏了攻擊者的真實身分。

嗅探
Sniffing
Packet sniffing captures sensitive network traffic.
封包嗅探擷取敏感的網路流量。


資訊安全是持續的攻防戰。對每一個輸入保持懷疑，對每一次傳輸加密，是保護使用者資料的不二法門。感謝收聽。
