歡迎來到行動開發篇。這集我們將比較 iOS 與 Android 兩大平台的專用術語與開發工具。如果你對 App 開發感興趣，這集是你的必修課。

行動開發的世界主要由兩大陣營組成，但也多了跨平台的選擇，接下來我們先認識這幾個主要名詞。

iOS
iOS 是 iPhone Operating System 的縮寫
iOS is the operating system for Apple's mobile devices.
iOS 是 Apple 行動裝置的作業系統。

Android
Android development requires understanding the Activity lifecycle.
Android 開發需要理解 Activity 的生命週期。

Swift
Swift is the modern language for building iOS apps.
Swift 是建構 iOS App 的現代語言。

Kotlin
Kotlin is the preferred language for Android development.
Kotlin 是 Android 開發的首選語言。

Flutter
Flutter allows you to build cross-platform apps from a single codebase.
Flutter 讓你從單一程式碼庫建構跨平台 App。

React Native
React Native uses JavaScript to create native mobile apps.
React Native 使用 JavaScript 來建立原生行動 App。

開發 App 時，接下來我們需要熟悉這些控制畫面與流程的核心組件。

視圖控制器
View Controller
The View Controller manages the screen's content in iOS.
View Controller 在 iOS 中管理螢幕內容。

活動
Activity
The Activity represents a single screen with a user interface in Android.
Activity 代表 Android 中具有使用者介面的單一螢幕。

片段
Fragment
A Fragment represents a reusable portion of the UI.
Fragment 代表 UI 中可重複使用的部分。

佈局
Layout
Use Auto Layout to support different screen sizes.
使用 Auto Layout 來支援不同的螢幕尺寸。

導覽
Navigation
Implement a navigation controller to manage screen transitions.
實作導覽控制器以管理螢幕轉換。

工欲善其事，必先利其器。熟練這些工具能讓你事半功倍，接著我們來看看有哪些好幫手。

模擬器
Simulator
Test your iOS app on the Xcode Simulator.
在 Xcode 模擬器上測試你的 iOS App。

仿真器
Emulator
The Android Emulator can be slow on older machines.
Android 仿真器在舊機器上可能會很慢。

介面建構器
Interface Builder
Use Interface Builder to design the UI visually.
使用 Interface Builder 以視覺化方式設計 UI。

軟體開發套件
SDK
SDK 是 Software Development Kit 的縮寫
Download the Android SDK to start developing.
下載 Android SDK 以開始開發。

建置工具
Gradle
Gradle is the build automation tool used in Android Studio.
Gradle 是 Android Studio 使用的自動化建置工具。

寫程式過程中，各種除錯手段與設計模式也是不可或缺，例如以下這幾個：

委派
Delegate
The TableView delegate handles row selection events.
TableView Delegate 處理列的選取事件。

除錯橋
ADB
ADB 是 Android Debug Bridge 的縮寫
Use ADB to install APKs manually.
使用 ADB 手動安裝 APK。

崩潰報告
Crash Report
Check the crash report to see why the app closed unexpectedly.
檢查崩潰報告以查看 App 為何意外關閉。

發布 App 前，還有許多設定與審查流程需要注意，接下來別忘了這些步驟。

資訊清單
Manifest
The Android Manifest file describes essential app information.
Android Manifest 檔案描述了重要的 App 資訊。

套件識別碼
Bundle ID
The Bundle ID uniquely identifies your app on the App Store.
Bundle ID 在 App Store 上唯一識別你的 App。

簽署
Signing
You must sign the app with a certificate before publishing.
發布前你必須使用憑證對 App 進行簽署。

審查
Review
The App Store review process can take a few days.
App Store 審查流程可能需要幾天時間。

推播通知
Push Notification
Send push notifications to engage users.
發送推播通知以與使用者互動。


行動開發有其獨特的魅力與挑戰。不管你是 iOS 派還是 Android 派，掌握這些術語將有助於你構建出色的應用程式。感謝收聽。
