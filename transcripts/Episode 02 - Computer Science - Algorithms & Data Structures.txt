歡迎來到演算法與資料結構篇。這是電腦科學中最基礎但也最重要的領域，也是程式設計的靈魂。無論是面試刷題還是效能最佳化，這些術語都是你展現硬實力的關鍵。

首先，我們從衡量程式效能的指標開始，了解如何評估程式碼的優劣。

時間複雜度
Time Complexity
Time complexity estimates how the runtime increases with the input size.
時間複雜度估算執行時間如何隨著輸入大小增加。

空間複雜度
Space Complexity
Space complexity measures the memory required by the algorithm.
空間複雜度測量演算法所需的記憶體。

大 O 符號
Big O Notation
Big O notation describes the worst-case performance of an algorithm.
大 O 符號描述演算法在最差情況下的效能表現。

常數時間
Constant Time
Accessing an array index takes constant time, or O(1).
存取陣列索引花費常數時間，即 O(1)。

線性時間
Linear Time
Iterating through a list is a linear time operation, or O(n).
迭代過一個列表是線性時間操作，即 O(n)。

對數時間
Logarithmic Time
Binary search runs in logarithmic time, or O(log n).
二元搜尋以對數時間執行，即 O(log n)。

資料結構是儲存與組織資料的方式，選擇正確的結構能大幅提升效率。

堆疊
Stack
A stack follows the First-In-Last-Out (LIFO) principle.
堆疊遵循先進後出 (LIFO) 原則。

佇列
Queue
A queue follows the First-In-First-Out (FIFO) principle.
佇列遵循先進先出 (FIFO) 原則。

鏈結串列
Linked List
A linked list consists of nodes referencing the next node.
鏈結串列由參考下一個節點的節點組成。

雜湊表
Hash Map
Use a hash map for fast lookups based on keys.
使用雜湊表來進行基於鍵值的快速查找。

二元樹
Binary Tree
In a binary tree, each node has at most two children.
在二元樹中，每個節點最多有兩個子節點。

圖
Graph
Graphs are used to model social networks and map routes.
圖被用來模擬社群網路與地圖路徑。

頂點
Vertex
A vertex represents a point or a node in a graph.
頂點代表圖中的一個點或節點。

邊
Edge
An edge connects two vertices in a graph.
邊連接圖中的兩個頂點。

最後，我們看看解決問題的核心技巧，也就是常見的演算法概念。

遞迴
Recursion
Recursion occurs when a function calls itself.
當函式呼叫自己時，就發生了遞迴。

迭代
Iteration
Use a loop for iteration to process the list items.
使用迴圈進行迭代以處理列表項目。

排序
Sorting
Quick sort is a highly efficient sorting algorithm.
快速排序是一種高效的排序演算法。

搜尋
Searching
Binary search only works on sorted arrays.
二元搜尋僅適用於已排序的陣列。

動態規劃
Dynamic Programming
DP solves complex problems by breaking them into simpler subproblems.
動態規劃 (DP) 透過將複雜問題分解為較簡單的子問題來解決。

深度優先搜尋
DFS
DFS 是 Depth-First Search 的縮寫
DFS explores as far as possible along each branch before backtracking.
DFS 在回溯之前，會沿著每個分支盡可能深入探索。

廣度優先搜尋
BFS
BFS 是 Breadth-First Search 的縮寫
BFS explores all neighbor nodes at the present depth first.
BFS 優先探索目前深度的所有鄰近節點。

暴力法
Brute Force
Brute force checks every possible solution, which is often slow.
暴力法檢查每一個可能的解，這通常很慢。

演算法與資料結構是區分「寫出程式」與「寫好程式」的分水嶺。掌握這些核心概念，不僅能讓你搞定面試，更能讓你在面對複雜系統時，做出最精準的設計決策。持續打磨這些基本功，你的技術底蘊將會越來越深厚。感謝收聽。
