歡迎來到電腦科學基礎篇。了解電腦如何運作以及網路如何傳輸，能讓你成為更全面的工程師。這集涵蓋了硬體組件與網路通訊協定的基礎知識。

首先，我們從電腦的核心硬體組件開始。

處理器
CPU
CPU 是 Central Processing Unit 的縮寫
The CPU executes instructions from the software.
CPU 執行來自軟體的指令。

核心
Kernel
The kernel is the core part of the operating system.
核心是作業系統的核心部分。

圖形處理器
GPU
GPU 是 Graphics Processing Unit 的縮寫
The GPU accelerates graphics rendering and machine learning tasks.
GPU 加速圖形渲染與機器學習任務。

記憶體
RAM
RAM 是 Random Access Memory 的縮寫
More RAM allows you to run more applications simultaneously.
更多 RAM 讓你能夠同時執行更多應用程式。

硬碟
Hard Drive
SSD hard drives are much faster than traditional HDDs.
SSD 硬碟比傳統 HDD 快得多。

位元組
Byte
A byte consists of 8 bits.
一個 Byte 由 8 個 Bit 組成。

接下來，我們進入網路連接的基礎概念，這些是建立連線的關鍵。

本機
Localhost
Localhost refers to the current computer you are using.
Localhost 指的是你目前正在使用的這台電腦。

IP 位址
IP Address
IP Address 是 Internet Protocol Address 的縮寫
Every device on the network has a unique IP address.
網路上的每個裝置都有一個唯一的 IP 位址。

網域名稱系統
DNS
DNS 是 Domain Name System 的縮寫
DNS translates domain names into IP addresses.
DNS 將網域名稱翻譯為 IP 位址。

連接埠
Port
Web servers typically listen on port 80 or 443.
網頁伺服器通常監聽 80 或 443 Port。

插座
Socket
A socket connection allows bi-directional communication.
Socket 連線允許雙向溝通。

協定
Protocol
TCP is a reliable transmission protocol.
TCP 是一種可靠的傳輸協定。

封包
Packet
Data is transmitted over the network in small packets.
資料在網路上以小封包的形式傳輸。

頻寬
Bandwidth
Streaming video requires high bandwidth.
串流影片需要高頻寬。

延遲
Latency
Low latency is crucial for online gaming.
低延遲對線上遊戲至關重要。

緩衝區
Buffer
The video is buffering because of the slow connection.
影片因為連線慢正在緩衝中。

編碼
Encoding
Ensure the file uses UTF-8 encoding.
確保檔案使用 UTF-8 編碼。

解碼
Decoding
The browser handles the decoding of the image data.
瀏覽器處理影像資料的解碼。

開發好的網頁應用程式需要透過伺服器傳遞，以下是幾種常見的網頁伺服器軟體。

IIS
IIS 是 Internet Information Services 的縮寫
IIS is a flexible web server developed by Microsoft.
IIS 是由微軟開發的彈性網頁伺服器。

Apache
Apache HTTP Server is a widely used open-source web server software.
Apache HTTP Server 是廣泛使用的開源網頁伺服器軟體。

Nginx
Configure Nginx as a reverse proxy to handle web traffic.
設定 Nginx 作為反向代理來處理網頁流量。

Kestrel
Kestrel is a cross-platform web server for ASP.NET Core.
Kestrel 是 ASP.NET Core 的跨平台網頁伺服器。

最後，我們來看看支撐整個網路運作的安全與基礎設施術語。

訊息佇列遙測傳輸
MQTT
MQTT 是 Message Queuing Telemetry Transport 的縮寫
MQTT is a lightweight messaging protocol for IoT devices.
MQTT 是一種用於物聯網裝置的輕量級訊息協定。

路由器
Router
The router directs traffic between different networks.
路由器在不同網路間引導流量。

交換器
Switch
The switch connects devices within the same local network.
交換器連接同一個區域網路內的裝置。

防火牆
Firewall
Configure the firewall to block unauthorized access.
設定防火牆以封鎖未經授權的存取。

虛擬私人網路
VPN
VPN 是 Virtual Private Network 的縮寫
Use a VPN to encrypt your internet traffic.
使用 VPN 來加密你的網際網路流量。

代理
Proxy
Use a proxy server to hide your original IP address.
使用代理伺服器來隱藏你的原始 IP 位址。

驅動程式
Driver
Install the latest driver for your graphics card.
為你的顯示卡安裝最新的驅動程式。

韌體
Firmware
Update the router firmware to fix security bugs.
更新路由器韌體以修復安全漏洞。


地基打得穩，大樓才蓋得高。了解這些知識，能幫助你從根本上理解系統效能與網路問題。無論技術如何更迭，這些核心原理始終不變。感謝收聽。
