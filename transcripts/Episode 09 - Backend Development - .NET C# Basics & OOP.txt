歡迎來到 .NET C# 基礎篇。這集我們聚焦於語言核心與物件導向概念。這些詞彙是微軟生態系開發者的基石，幫助你精確描述程式碼結構。

我們從 C# 專案的組織結構開始看起，了解程式碼是如何被管理的。

解決方案
Solution
Open the solution file in Visual Studio to see all projects.
在 Visual Studio 中開啟解決方案檔以查看所有專案。

組件
Assembly
The compiler generates an assembly like a DLL or EXE file. DLL stands for Dynamic Link Library.
編譯器會產生像 DLL 或 EXE 這樣的組件檔案。其中 DLL 代表動態連結程式庫。

命名空間
Namespace
Use the correct namespace to organize your classes logically.
使用正確的命名空間來邏輯性地組織你的類別。

相依性
Dependency
Update the NuGet dependency to the latest version to fix bugs.
將 NuGet 相依套件更新到最新版以修復 Bug。

在建立軟體時，我們需要定義各種型別，接下來是 C# 中最基本的型別定義。

類別
Class
Define a class to represent the blueprint of an object.
定義一個類別來代表物件的藍圖。

結構
Struct
A struct is a value type, unlike a class which is a reference type.
結構是實值型別，不同於類別是參考型別。

介面
Interface
The service class must implement the IService interface.
服務類別必須實作 IService 介面。

列舉
Enum
Use an enum to define the status options clearly.
使用列舉來清楚定義狀態選項。

委派
Delegate
Pass a method as a delegate parameter to handle the callback.
將方法當作委派參數傳遞以處理回呼。

在類別內部，我們接著介紹各式各樣的成員來定義行為與資料。

屬性
Property
Use properties with getters and setters instead of public fields.
使用具備 Get 和 Set 的屬性而非公開欄位。

欄位
Field
A private field should be named with an underscore prefix.
私有欄位應該以底線前綴命名。

建構子
Constructor
Inject required services through the class constructor.
透過類別建構函式注入所需的服務。

方法
Method
The method performs a specific task or calculation.
方法執行特定的任務或計算。

靜態
Static
Static members belong to the class itself, not to specific instances.
靜態成員屬於類別本身，而非特定實例。

唯讀
Read-only
Read-only fields can only be assigned in the declaration or constructor.
Read-only 欄位只能在宣告或建構函式中賦值。

多載
Overload
Method overloading allows creating methods with the same name but different parameters.
方法多載允許建立同名但參數不同的方法。

物件導向程式設計有三大核心支柱，這些概念讓我們的程式碼更具彈性與可維護性，讓我們接著往下看。

封裝
Encapsulation
Encapsulation hides the internal state and exposes only necessary methods.
封裝隱藏了內部狀態，只暴露必要的方法。

繼承
Inheritance
The class uses inheritance to share logic from the base class.
這個類別使用繼承來共享基底類別的邏輯。

多型
Polymorphism
Polymorphism allows us to treat different objects in the same way.
多型讓我們能以相同的方式對待不同的物件。

為了實作繼承與多型，我們會用到接下來這些修飾關鍵字。

抽象
Abstract
Define an abstract method that all derived classes must implement.
定義一個所有衍生類別都必須實作的抽象方法。

虛擬
Virtual
You can override a virtual method in the derived class.
你可以在衍生類別中覆寫虛擬方法。

覆寫
Override
Override the base method to provide a custom implementation.
覆寫基底方法以提供自訂的實作。

密封的
Sealed
A sealed class cannot be inherited by other classes.
密封類別無法被其他類別繼承。

接下來是定義程式碼可見範圍與封裝層級的存取修飾詞。

公開的
Public
Public members are accessible from anywhere in the code.
公開成員可以在程式碼的任何地方存取。

私有的
Private
Private members are only accessible within the declaring class.
私有成員只能在宣告的類別內部存取。

受保護的
Protected
Protected members are accessible within the class and its subclasses.
受保護成員可以在類別及其子類別中存取。

內部的
Internal
Internal members are accessible only within the same assembly.
內部成員只能在同一個組件內存取。

在程式執行過程中，我們需要處理錯誤並管理流程，以下是相關的控制語法。

例外
Exception
Catch the exception in a try-catch block to prevent a crash.
在 Try-Catch 區塊中捕捉例外以防止當機。

拋出
Throw
Throw an argument exception if the input is invalid.
如果輸入無效，則拋出參數例外。

非同步
Asynchronous
Asynchronous operations prevent the UI from freezing. UI stands for User Interface.
非同步操作可防止使用者介面凍結。UI 代表使用者介面。

等待
Await
Await the async task result before proceeding to the next line.
在繼續執行下一行前，先等待非同步 Task 的結果。

執行緒
Thread
Do not block the main thread with heavy computations.
不要用繁重的運算阻塞主執行緒。

接下來是構成程式邏輯的基礎語句，包含變數宣告與運算。

變數
Variable
Initialize the variable with a default value before using it.
使用變數前先用預設值初始化。

字串
String
Strings are immutable in C#, so modifying one creates a new object.
在 C# 中字串是不可變的，所以修改它會建立一個新物件。

整數
Integer
Parse the user input string to an integer for calculation.
將使用者輸入的字串解析為整數以進行計算。

運算子
Operator
Use the null-coalescing operator to handle null values gracefully.
使用 Null 合併運算子來優雅地處理 Null 值。

可為空的
Nullable
This property is nullable because the database field allows nulls.
這個屬性是可為空的，因為資料庫欄位允許 Null。

轉型
Cast
You can cast the object to the specific interface to access its methods.
你可以將該物件轉型為特定介面以存取其方法。

參數
Parameter
The method definition includes two string parameters.
這個方法定義包含了兩個字串參數。

引數
Argument
Pass the correct arguments when calling the function.
呼叫函式時傳遞正確的引數。

程式寫完後，我們需要經過編譯與執行的過程，這些術語描述了程式碼的生命週期。

編譯
Compile
The code fails to compile due to a syntax error on line 10.
程式碼因第 10 行的語法錯誤而無法編譯。

建置
Build
The CI pipeline failed during the build step. CI stands for Continuous Integration.
CI 管線在建置步驟失敗了。CI 代表持續整合。

執行階段
Runtime
Some errors, like null references, only appear at runtime.
有些錯誤，如 Null 參考，只會在執行階段出現。

除錯
Debug
Attach the debugger to the process to inspect variables.
將除錯器附掛到程序以檢查變數。


.NET 的世界博大精深，但這些基礎詞彙是通往高階架構的必經之路。掌握了這些，你就能更精確地描述程式碼的行為。感謝收聽。
