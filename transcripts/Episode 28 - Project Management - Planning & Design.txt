歡迎來到專案規劃與設計篇。軟體開發不是一個人的事。這集整理了與產品經理及設計師在規劃階段溝通時，最常出現的關鍵字。

專案啟動時，確認範圍與目標是首要任務，我們先看規劃階段的關鍵字。

利害關係人
Stakeholder
We need to update the stakeholders on the project progress.
我們需要向利害關係人更新專案進度。

產品路線圖
Roadmap
Check the roadmap to see upcoming features for next year.
查看路線圖以了解明年即將推出的功能。

里程碑
Milestone
We have reached a major milestone by completing the backend.
完成後端開發讓我們達到了一個主要的里程碑。

截止日期
Deadline
The deadline is next Friday, so we need to hurry.
截止日期是下週五，所以我們要快一點。

時間軸
Timeline
The project timeline is very tight given the scope.
考慮到範圍，專案的時間軸非常緊湊。

季度
Quarter
Q4
Fourth Quarter
This is our main goal for Q4.
這是我們第四季的主要目標。

需求定義不清，是專案失敗的主因，因此我們需要明確的規格術語。

規格
Spec
Spec 是 Specification 的縮寫
The feature spec is not finalized, so requirements might change.
功能規格還沒定案，所以需求可能會變。

用戶故事
User Story
"As a user, I want to login" implies a user story.
「作為使用者，我想要登入」暗示了一個用戶故事。

驗收標準
Acceptance Criteria
The feature must meet all acceptance criteria before QA approves.
功能必須符合所有驗收標準，QA 才會核准。

可交付成果
Deliverable
What are the key deliverables for this sprint?
這個衝刺的關鍵可交付成果是什麼？

最小可行性產品
MVP
MVP 是 Minimum Viable Product 的縮寫
We are launching an MVP to test the market reaction.
我們正推出 MVP 來測試市場反應。

範圍
Scope
This request is out of scope for the current phase.
這個請求超出了目前階段的範圍。

設計階段將需求轉化為可見的藍圖，常見的產出物如下。

線框圖
Wireframe
The designer provided the wireframes for the new layout.
設計師提供了新版面的線框圖。

視覺稿
Mockup
The high-fidelity mockup shows exactly how the app will look.
高保真視覺稿展示了 App 確切看起來會是什麼樣子。

原型
Prototype
Build a quick prototype to validate the user interaction.
建立一個快速原型來驗證使用者互動。

回饋
Feedback
We received positive feedback from the beta testers.
我們收到來自 Beta 測試者的正面回饋。

迭代
Iteration
We improve the product quality in every iteration.
我們在每次迭代中改進產品品質。

上線與維運是檢驗成果的時刻，這些術語涵蓋了解決最後一哩路的過程。

部署
Deploy
We will deploy the hotfix to production tonight.
我們今晚會部署 Hotfix 到正式環境。

回滾
Rollback
We had to rollback the deployment due to a crash.
因為當機，我們必須回滾這次部署。

變更紀錄
Changelog
Check the changelog for details on what's new.
查看變更紀錄以了解更新細節。

功能切換
Feature Flag
Use a feature flag to verify the code in production safely.
使用功能切換來安全地在正式環境驗證程式碼。

維護
Maintenance
The system is under scheduled maintenance for upgrades.
系統正在進行排程維護以進行升級。

資源
Resource
We lack the engineering resources to build this now.
我們現在缺乏工程資源來打造這個。

指標
Metric
User engagement is a key business metric we track.
使用者參與度是我們追蹤的關鍵商業指標。


與非技術人員溝通時，使用這些標準術語能有效減少誤解，確保大家都在同一個頻率上。感謝收聽。
