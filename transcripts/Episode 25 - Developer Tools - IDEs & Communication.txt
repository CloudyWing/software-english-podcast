歡迎來到開發工具篇。除了寫程式，選擇正確的工具能讓效率翻倍。這集我們介紹編輯器、專案管理平台以及常用的通訊軟體，這些都是開發者每天必備的利器。

工欲善其事，必先利其器，我們先從編輯器與 IDE 開始，了解它們的功能。

整合開發環境
IDE
IDE 是 Integrated Development Environment 的縮寫
Visual Studio is a powerful IDE for .NET developers.
Visual Studio 是給 .NET 開發者的一款強大整合開發環境。

Visual Studio Code
VS Code
VS Code is a lightweight but powerful source code editor.
VS Code 是一個輕量但強大的原始碼編輯器。

擴充套件
Extension
Install the Prettier extension to format your code automatically.
安裝 Prettier 擴充套件來自動格式化你的程式碼。

主題
Theme
I prefer a dark theme to reduce eye strain.
我偏好深色主題以減少眼睛疲勞。

程式碼片段
Snippet
Use a code snippet to quickly insert a for-loop structure.
使用程式碼片段快速插入 For 迴圈結構。

快捷鍵
Shortcut
Learn the keyboard shortcuts to speed up your coding.
學習鍵盤快捷鍵以加快你的寫程式速度。

隨著 AI 技術進步，新一代的 AI 編輯器正在改變我們的開發模式，接下來讓我們看看這些強調與 AI 深度整合的工具。

Cursor
Cursor is an AI-first code editor built for pair programming with AI.
Cursor 是一個為了與 AI 結對程式設計而打造的 AI 優先程式碼編輯器。

Windsurf
Windsurf integrates deep context awareness for better AI coding suggestions.
Windsurf 整合了深度上下文感知，以提供更好的 AI 程式碼建議。

Antigravity
Antigravity is a powerful internal AI agentic coding assistant.
Antigravity 是一個強大的內部 AI 代理程式設計助理。

GitHub Copilot
GitHub Copilot suggests whole lines or entire functions right inside your editor.
GitHub Copilot 直接在你的編輯器中建議整行或完整的函式。

氛圍開發
Vibe Coding
Vibe Coding refers to a flow state where you focus on high-level logic while AI handles the implementation.
Vibe Coding 指的是一種專注於高層次邏輯，而將實作交給 AI 的心流狀態。

每個開發者的工具箱裡，都少不了這些實用的小工具，接下來介紹的這些能幫你解決各種疑難雜症。

Docker Desktop

Stack Overflow
