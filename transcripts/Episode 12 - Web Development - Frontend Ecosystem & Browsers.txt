歡迎來到前端生態系篇。這一集我們聚焦於瀏覽器技術與現代框架。Chrome 不僅是瀏覽器，更是開發工具；而 React 與 TypeScript 則是現代前端的標準配備。

開發網頁的第一步，是了解我們的戰場——瀏覽器，以下是它的核心組成。

Chrome
Google Chrome is the most widely used browser for web development.
Google Chrome 是網頁開發最廣泛使用的瀏覽器。

V8 引擎
V8 Engine
Node.js is built on Chrome's V8 JavaScript engine.
Node.js 是建立在 Chrome 的 V8 JavaScript 引擎之上。

Firefox
Make sure to test the CSS layout on Firefox as well.
務必確認 CSS 版面配置在 Firefox 上也正常。

Safari
Safari on iOS requires specific meta tags for proper display.
iOS 上的 Safari 需要特定的 Meta 標籤才能正確顯示。

開發者工具
DevTools
Open Chrome DevTools to inspect the HTML elements and console.
開啟 Chrome DevTools 來檢查 HTML 元素與控制台。

控制台
Console
Log the error message to the console for debugging.
將錯誤訊息記錄到控制台以進行除錯。

現代前端開發離不開強大的建置與語言工具，這些能提升開發效率。

TypeScript
TypeScript adds static typing to JavaScript, reducing runtime errors.
TypeScript 為 JavaScript 增加了靜態型別，減少了執行階段錯誤。

Node.js
Use Node.js to run JavaScript on the server side.
使用 Node.js 在伺服器端執行 JavaScript。

NPM
NPM 是 Node Package Manager 的縮寫
Install the package using NPM.
使用 NPM 安裝這個套件。

打包工具
Bundler
Webpack is a module bundler that compiles your JavaScript.
Webpack 是一個編譯你 JavaScript 的模組打包工具。

框架能幫助我們更有效率地建構複雜的使用者介面，最熱門的三大框架如下。

React
React uses a virtual DOM to optimize rendering performance.
React 使用虛擬 DOM 來最佳化渲染效能。

Vue
Vue is known for its gentle learning curve and flexibility.
Vue 以平緩的學習曲線與靈活性著稱。

Angular
Angular is a comprehensive framework maintained by Google.
Angular 是由 Google 維護的全方位框架。

Next.js
Next.js is a React framework that supports server-side rendering.
Next.js 是一個支援伺服器端渲染的 React 框架。

選擇合適的渲染架構，能大幅提升使用者體驗，我們來看看各種渲染模式。

單頁應用程式
SPA
SPA 是 Single Page Application 的縮寫
React is a popular framework for building interactive SPAs.
React 是建構互動式 SPA 的熱門框架。

伺服器端渲染
SSR
SSR 是 Server-Side Rendering 的縮寫
SSR improves SEO and initial load performance.
SSR 改善了 SEO 和初始載入效能。

客戶端渲染
CSR
CSR 是 Client-Side Rendering 的縮寫
CSR offloads rendering work to the user's browser.
CSR 將渲染工作交給使用者的瀏覽器。

響應式
Responsive
The web design must be responsive for mobile and tablet devices.
網頁設計必須對手機和平板裝置具響應性。

處理資料儲存與即時通訊，也是前端的重要職責，相關技術包括：

本機儲存
LocalStorage
Save the user preference in LocalStorage so it persists after refresh.
將使用者偏好儲存在 LocalStorage，這樣重新整理後還會在。

WebSocket
Use WebSockets for real-time features like chat applications.
使用 WebSockets 開發如聊天應用程式等即時功能。


前端技術更新迭代非常快，但瀏覽器與 JavaScript 的核心原理始終不變。掌握這些基礎，你就能輕鬆應對任何新框架。感謝收聽。
